总结

每个模板经过 compile 都会生成一个 render 函数

render 作为 渲染三部曲的第二部，主要作用就是 执行 render，生成 Vnode

把 template 上绑定的数据，都保存到 vnode 中

然后，生成 Vnode，就是为了给 渲染三部曲的 第三部 Diff 提供源动力

从而完成 DOM 挂载
