《央视融媒》视频上传解决方案

Author: Wenng Date: December 17, 2019  17:53:08

自我介绍：

我是我部负责技术的一名员工，本篇文章主要解决视频上传速度的解决办法。

一.邮箱上传

首先以QQ邮箱为例子，只要使用发送邮件即可，添加附件。

但是这个上传的速度非常感人。

二.百度云上传

这个方法是基于第三方平台的，百度网盘，其实某度做了这么多人，搜索引擎被人骂的体无完肤，也在乡亲父老们口上留了一句话，有问题上百度，看某度，某度的第一条新闻肯定是跑题的，这也就是为啥骂他的原因，简直就是收钱办事，说了这么多但是百度网盘的效率还是蛮不错的，虽然都要钱把，但是这个上传下载的速度还是比邮箱稍微快一点的。

优点：

  1. 视频云存储。
  2. 一次上传N次下载。
  3. 视频分享便捷。

方法步骤：

  1. 注册百度账号，下载百度网盘。

    PC端地址：http://pan.baidu.com/download

    APP端：直接打开应用商店下载即可。

  2. 登录客户端主页面使用上传的功能。

  3. 将视频制作为链接进行分享。

  4. 就可以将这个链接获取到，在使用邮箱将链接，和提取码发送给我部我部会将链接内容提取下载。

  5. 普通用户有两个T的存储空间所以不要在意空间限制。

  6. 这就是视频上传的基本方法。
