捋羊毛，捋死羊！！

Author: Wenng Date: November 7, 2019  20:36:35

今天出去参加面试回来打开知乎想看看热榜新闻是啥，其实八卦的咱也不关注，技术问题，社会新闻咱也会看一看，接下来我描述一下情况。

如图这边是今天的知乎热榜，这还少淘宝官方今天中午发的帖子。

事件是因为一个主播引起的轰动粉丝对出现BUG的卖家疯狂刷单，最后导致无法发货然后举报，获取一定的补偿金，甚至导致店铺关门，负债累累。。

可以定义为网络暴力！还是那种有利可图的网络暴力！

上面是这个诱导者道歉的一条信息，但是在我看了整个帖子之后，我发现他做这个事情已经不是一次两次了，专业捋羊毛，

捋羊毛

他在网络暴力别人的同时也让自己陷入了泥潭，最后发现是一个沼泽。

作者：拥抱风

他并不是在赔偿，他只是在低头。

他最害怕的是这件事情闹大，结果他最担心的还是发生了。

他没有类似的公关经验，他担心名声恶臭，无法继续赚钱。他唯一能想到的是赔钱，赔多少合适又显得有诚意而不敷衍呢？

一开始的退款申明，再到后来的赔偿两万，为什么是两万呢，或许这是他一个月视频的基本收入，他想的是我一个月的盈利应该够了吧，有诚意了吧，这些多管闲事的人能放过我了吧。结果往往事与愿违。为什么这些人越来越愤怒了呢?那我赔偿多一点，赔多少合适呢？查了下开店的费用，13万是封顶的费用，我赔偿这个合适了吧，我已经这么有诚意了，你们怎还不放过我呢？

我不相信他是不知道一个淘宝店做起来是需要很多推广费用的，还有很多的心血消耗，这些他都明白，为什么他只字不提呢？因为他在计较得失，他知道太贵了。所以只好装傻，博取同情，赔偿店主，发出声明。

这一切他都只是在给愤怒的舆论低头，他一定很惊慌，舆论像只凶狼不仅要吃掉他的血肉，要把他的骨头嚼碎，还要把他的皮囊钉在耻辱柱上，最后用滚烫的油反复泼烫。

这很残忍，也不人道，但是又怎么样呢？当时那么骄傲的他给他粉丝看他的战利品的时候本就是恶魔，是他一手造就了这个悲剧，是他亲手打开了人性的恶。

舆论也会让他见识人性善的一面，只是这种人性的善对他这样的人有点残忍，像是道士手中的桃木剑，像是刽子手给犯人千刀万剐的那把刀。它从不想给予他这种人改过的机会，它想用最残忍的方式凌迟他，它不可控，它也不太理智，甚至有点愚蠢，但是它是对待他这种法律无法严惩的凶徒最好的方式。

给人造成的伤害无法弥补，那只好让他这种人付出更重的代价。

可以卑微如尘土 不可扭曲如蛆虫。
